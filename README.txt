Extend Post and Page data with all related meta data.

== Description ==

Automatically extends Post and Page data with an array containing all of the extra meta data pertinent to the `$post` object.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The `$post` object will contain a sub-object `$post->metadata` after running a regular WordPress query, containing any additional meta information assigned to the Post.
